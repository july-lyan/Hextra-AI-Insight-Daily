## 📋 配置步骤（按顺序执行）

### 步骤 1: 更新远程仓库地址（可选）

如果您想将代码推送到您自己的 Fork 仓库，请告诉我您的 GitHub 用户名，我可以帮您更新。

或者您也可以手动运行：

```bash
# 将 YOUR_USERNAME 替换为您的 GitHub 用户名
git remote set-url origin https://github.com/YOUR_USERNAME/Hextra-AI-Insight-Daily.git
```

### 步骤 2: 配置 GitHub Pages ⚠️ 最重要

1. **打开您的 Fork 仓库**
   - 访问：`https://github.com/YOUR_USERNAME/Hextra-AI-Insight-Daily`
   - 将 `YOUR_USERNAME` 替换为您的 GitHub 用户名

2. **进入 Settings**
   - 点击仓库顶部的 **Settings** 标签页

3. **配置 Pages**
   - 左侧菜单找到 **Pages**
   - **Source** 选择：**GitHub Actions** ✅
   - **不要**选择 "Deploy from a branch" ❌
   - 点击 **Save**

### 步骤 3: 配置 APP_BASE_URL 变量（推荐）

1. **Settings** -> **Secrets and variables** -> **Actions** -> **Variables**
2. 点击 **New repository variable**
3. 添加：
   - **Name**: `APP_BASE_URL`
   - **Value**: `https://YOUR_USERNAME.github.io/Hextra-AI-Insight-Daily`
   - 将 `YOUR_USERNAME` 替换为您的 GitHub 用户名
4. 点击 **Add variable**

### 步骤 4: 触发首次部署

**方式 1: 手动触发（推荐）**
1. 进入 **Actions** 标签页
2. 选择 **Deploy Hugo site to Pages**
3. 点击 **Run workflow** -> **Run workflow**

**方式 2: 推送代码触发**
```bash
git add .
git commit -m "docs: 添加配置文档"
git push origin main
```

### 步骤 5: 验证部署

1. 等待 2-5 分钟
2. 访问：`https://YOUR_USERNAME.github.io/Hextra-AI-Insight-Daily`
3. 确认网站可以正常访问
